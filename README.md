#  News Topic Classifier Using BERT

A fine-tuned **BERT-base-uncased** transformer model that classifies news headlines into four categories: World, Sports, Business, and Sci/Tech.

##  Live Demo
Check out the live model here: [Hugging Face Space](https://huggingface.co/spaces/MuhammadSaad1234/News-Classifier-BERT)

##  Project Overview
- **Dataset:** AG News (Hugging Face Datasets)
- **Model:** `bert-base-uncased`
- **Metrics:** Achieved ~94% Accuracy and F1-Score.
- **Frameworks:** Transformers, PyTorch, Gradio.
